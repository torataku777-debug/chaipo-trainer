本プライバシーポリシーは、虎谷拓(以下「開発者」)が提供するモバイルアプリケーション「OFCネイバーズ」(以下「本アプリ」)におけるユーザー情報の取り扱いについて定めるものです。本アプリをご利用いただく前に、本ポリシーをお読みください。

1. 収集する情報について

本アプリは、ユーザーから個人を特定できる情報を直接収集することはありません。ただし、 機能の一部(フレンド対戦機能)では Supabase をバックエンドサーバーとして利用するため、 該当機能の利用時には以下の情報が外部サーバーに保存されます。

1-A. 端末内(ローカルストレージ)にのみ保存される情報

  • アプリ設定情報(制限時間、初期点数、その他のゲーム設定)
  • プレイログ・対戦履歴(AI 対戦・ファンタジーランド練習)
  • ファンタジーランド練習に関する統計
  • シーズン開始時刻
  • その他ゲーム進行に必要な情報

これらのデータは、お使いの端末を離れることはありません。

1-B. フレンド対戦機能利用時に外部サーバー (Supabase) に保存される情報

本アプリのフレンド対戦機能 (4桁コード共有によるオンライン対戦) を利用する際、 以下の情報が Supabase (Supabase Inc. が提供するクラウドサービス) のサーバーに保存されます。

  • 匿名ユーザー ID (Supabase が自動生成、 個人を特定できない UUID)
  • アプリが自動生成する表示名 (例: "Player-426e73"、 ユーザーが変更可能)
  • 対戦データ (4桁ルームコード、 配置したカード、 ゲーム進行状況、 スコア、 ファンタジーランド状態)
  • マッチ作成日時・終了日時・ステータス (waiting / playing / finished / abandoned)

これらの情報は対戦相手と同期するために必要なものであり、 開発者は商業目的でこれを利用しません。

2. 第三者への情報提供

本アプリは、ユーザーのいかなる情報も第三者(他のユーザー・企業・団体・個人を含む)に販売・提供することはありません。ただし、 フレンド対戦機能を利用する際は、 対戦相手 (4桁コードを共有した相手) に対して以下の情報が表示されます。

  • 表示名 (例: "Player-426e73"、 ユーザーが変更可能)
  • 対戦中の配置・スコア (ゲーム進行に必要な情報)

対戦相手以外の第三者には、 上記の情報は提供されません。

3. 広告・アクセス解析・クラッシュレポートについて

本アプリは、以下のサービス・SDKを一切使用しておりません。

  • 広告配信SDK(AdMob、その他)
  • アクセス解析・行動分析ツール(Google Analytics、Mixpanel、その他)
  • クラッシュレポートツール(Firebase Crashlytics、Sentry、その他)
  • ユーザートラッキング(IDFA等)

4. データの保管期間と削除

4-A. 端末内データの削除

  • 本アプリの「設定」画面内のデータ管理機能から削除する
  • 本アプリをアンインストールすることで、関連する全てのデータが削除されます

4-B. Supabase に保存されたデータの削除

フレンド対戦の対戦データ (matches テーブル) およびユーザーレコードは、 ユーザーから明示的な削除要請があった場合に開発者が手動で削除します。 削除を希望される場合は、 下記のお問い合わせ先までご連絡ください。

また、 終了したマッチデータ (status=finished/abandoned) は、 一定期間 (現時点では 90日) 経過後に自動削除される予定です。

5. 子供のプライバシー

本アプリは、年齢制限を設けず一般向けに提供されています。本アプリは個人を特定できる情報の収集を行わないため、 保護者の同意なくお子様にもご利用いただけます。 ただし、 フレンド対戦機能を利用する際は、 対戦相手と表示名や対戦データが共有されることをご理解の上、 保護者の方の判断のもとでご利用ください。

6. アプリ内課金について

本アプリ(v1.1時点)は、アプリ内課金機能を提供しておりません。将来的にアプリ内課金が導入される場合は、本ポリシーを改定のうえ、課金に関する情報の取り扱いを明記します。

7. 利用しているサードパーティサービス

本アプリの一部機能では、 以下のサードパーティサービスを利用しています。

  • Supabase(https://supabase.com/privacy): フレンド対戦のバックエンド (認証・データ同期) を提供。 Supabase 自身のプライバシーポリシーは上記リンクを参照してください。

8. プライバシーポリシーの変更

本ポリシーは、法令の改正・サービス内容の変更等に応じて予告なく変更される場合があります。重要な変更を行う際は、本ページおよびアプリ内にて事前に告知します。本ポリシーが改定された場合、改定後のポリシーが適用されます。

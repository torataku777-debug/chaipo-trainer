## 改善ポイント(★優先度順)

### ★★★ 1. プロフィールカードのヒーロー感強化

現状: 白カードに金 radial highlight が右下に控えめにあるが、その他のセクションカードとの差異が弱い。「ここがあなたのプロフィール」のピン留め感が足りない。

具体策:
- カード背景に **左上の白光 + 右下の金光** の二段 radial を強化(現状の `--profile-gold-glow` を 2 段重ね、半径と濃度を上げる)
- カード上端に **微細な金ハイライト**(`inset 0 1px 0` を金寄りに、もしくは追加の inset 線)
- アバター周辺に **微細な金キラ** 3-4 個 (SVG 星粒、脈動アニメ任意・静的でも可)
- カード自体に **わずかな浮き上がり感**(現状の box-shadow を 1 段強める)
- 段位バッジ「プラチナ 3」を **若干大きく+内側にスター光沢**(現状の星 SVG をやや大きく、内側にハイライト)

### ★★★ 2. 称号バッジ「FL職人」の色調整

現状: 紫 (`#5b21b6` / 背景 `#ede9fe→#ddd6fe`)。他のパレット(sky/pomu/leaf-green)と整合せず浮いている。

具体策(以下のいずれか):
- A. **金寄り**: `--pomu-yellow` 系の薄金グラデ + 深ブラウン文字。ただし段位バッジと色が被るので注意
- B. **深緑寄り**: `--leaf-green` 系の薄緑グラデ + 深緑文字(`#15803d`)
- C. **シアン寄り**: `--sky-300` 系の薄シアングラデ + 深紺文字
- 推奨は **C のシアン寄り**(プラチナ金・レート白・称号シアンで「金/白/シアン」の三色が綺麗に並ぶ)
- 紫グラデは完全除去

### ★★ 3. サウンド設定の追加(設定セクション)

現状: 4項目(時間制限/初期点/ハプティック/カードデザイン)。サウンドが欠けている。

具体策:
- 「ハプティック」と「カードデザイン」の間に **「サウンド」** 行を追加
- アイコン: スピーカー(SVG)
- 右側: トグル ON/OFF (既存の `.toggle-switch` 流用、`--leaf-green` で ON)
- サブテキスト: 「BGM・効果音」

(BGM音量・SE音量の独立スライダーは将来仕様。今回は1トグルで十分)

### ★★ 4. 通知設定の追加(設定セクション)

現状: 通知関連の項目なし。

具体策:
- 「サウンド」の下に **「通知」** 行を追加
- アイコン: ベル(SVG)
- 右側: トグル ON/OFF
- サブテキスト: 「対戦・お知らせ・無料パック」

設定セクションは 4 → 6 項目に拡張(時間制限/初期点/ハプティック/サウンド/通知/カードデザイン)。

### ★★ 5. プロフィールカードのレイアウト微調整

現状: 名前の右に編集アイコン、マイコードの右にコピーアイコン。視覚的に少し密集気味。

具体策:
- 編集アイコンを **少し離す**(現状 gap 6px → 8px)
- マイコードラベル「マイコード」を **uppercase tracking 強め**(現状 0.10em → 0.14em)で副情報感を明確化
- コピーアイコンに **ホバー時の薄いハイライト円**(`background: rgba(143,200,224,0.12); border-radius: 50%`)

### ★ 6. シェブロン濃度

現状: `--ink-300` (#d1d5db) で薄め。

具体策:
- `--ink-300` → `--ink-500` (#6b7280) もしくは中間 (#9ca3af) に変更
- iOS設定アプリ並みの視認性を確保

### ★ 7. ログアウトボタンのガラス感

現状: 白背景 + 赤テキスト + 薄い赤縁。

具体策:
- 背景に **微細なガラスグラデ**(`linear-gradient(180deg, #fff 0%, #fef2f2 100%)`)
- 上端に **inset highlight** で立体感
- 押下時の沈み込みを若干強める

### ★ 8. 行アイコンの統一感

現状: 一部のアイコン(初期点の時計、カードデザインのカード重ね、ランキングのトロフィー)の SVG ストロークが他と比べてやや雑。

具体策:
- 全 SVG の stroke-width を `2` に統一(視覚バランス)
- 各アイコンの内側描画密度を調整(過密でも過疎でもない)
- 特に **ランキングのトロフィー**: 取っ手の表現を整える
- **カードデザインのカード重ね**: 2枚のカードに少しスーツマーク(♠♥など極小)を追加して「カード」感を出す

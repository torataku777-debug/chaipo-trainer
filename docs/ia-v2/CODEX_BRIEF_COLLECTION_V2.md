## 改善ポイント(★優先度順)

### ★★★ 1. フィルタチップを画面幅に収める

現状: 「すべて / 背景 / フレーム / アバター / エフェクト」の5チップで、`エフェクト` が画面右端で見切れる(横スクロール可能だが見栄えが悪い)。

具体策:
- チップ padding を `7px 14px` → `6px 10px` 程度に縮める
- フォントサイズ 12px → 11px、letter-spacing を縮小
- gap 6px → 4-5px
- もしくは **5チップを均等幅 grid** にする(横スクロール撤廃、全幅 = 353px の中で完結)
- アクティブ状態: 現状の `--ink-900` 黒背景 → **`--pomu-yellow` → `--pomu-orange` の縦グラデ** + 白文字 にしてぽむタウンの色味と整合
- 非アクティブのチップに微細な内側ハイライト

### ★★★ 2. 装備中ストリップの「特別感」を強化

現状: 4枠の金枠 + 控えめなグロー。やや味気ない。

具体策:
- ストリップ全体のカード地に、ガチャ v3 の `is-ready` パックと同じ **金グロー外周**(`--ready-glow-soft`)を控えめに加える
- `EQUIPPED · 装備中` ラベルを **ピル型のゴールドバッジ**化(渋谷ナイトパックの `SEASON 1 LIMITED` ラベル同等の質感)
- 各スロットフレーム:
  - 上端の白ハイライトを強める(`--shop-card-highlight` 並みの存在感)
  - 内側に微細なソフト影で立体感
  - ぽむアバタースロットだけ薄い空ブルー背景にして他と差別化
- スロット名(青空・シンプル白等) の上に **「装備中」マイクロラベル**(オレンジドット + ink-500 の小文字) を入れる案も可

### ★★ 3. コレクションセル装備中状態(`.is-equipped`)の金グロー強化

現状: 金枠 + 軽い金グロー。ガチャ v3 の `is-ready` パックカードと比べて控えめすぎる。

具体策:
- `box-shadow` に **`0 0 18px var(--ready-glow-soft)`** を追加してパックの `is-ready` と整合
- 上端 4-6px に **薄い金光沢バー**(linear-gradient) を載せる
- ✓バッジ(右上)を **やや大きく(20→22px)**、光沢グラデと内側ハイライト追加
- 装備中セルに **微細な揺れアニメ** (`gachaSway` 同等、±0.6deg / 4s周期) を加える案も可。控えめに留めること。

### ★★ 4. 春の桜サムネを季節感のあるイラストに

現状: 上にピンクの円が散らされ、下に山シルエット。「桜の山」感が薄く、季節を感じにくい。

具体策:
- 桜の **花弁(5枚一組の円)** を空中に散らせる(現状の単色円 → ピンクの花弁シルエット)
- 下部の山シルエットを少し低くして、**桜の木のシルエット**(幹+冠の塊)を1-2本足す
- ピンクのグラデーションを **やや淡く**(現在の `#fbcfe8` 下端は濃すぎ)、空気感を出す

### ★★ 5. エフェクトサムネを機能名と一致させる

現状:
- **キラキラ**: 大きな黄色の星 + 周辺に小さな星3つ。良い。
- **リング**: 同心円ゴールド + 破線。「リング」が何のエフェクトか説明力不足。

具体策:
- **リング**: 「カード配置時に出る光輪」をイメージできるよう、**水紋風の同心円**(複数リング、外側がフェード) + 中央に金色ドット
- キラキラはほぼ維持で良いが、星の周りの光のオーラ(`--cEfKiGlow`)をやや強めて魔法陣感を増す

### ★ 6. 進捗フッターの存在感UP

現状: `COLLECTION 38 / 120 種` + 黄色グラデの 31.6% バー。控えめ。

具体策:
- 数字「38」を **大きく**(14→18px)、`/120 種` は小さく
- 進捗バーに **内側のシャイン**(上端の白ハイライト)を強める
- バーの右端に **小さな星 or キラ** マークを置いて到達感(任意)
- フッター全体に **微細な金グロー外周**(現状の `inset` 影だけでなく外側にも)

### ★ 7. 未取得セル(`.is-locked`)の鍵オーバーレイ精度向上

現状: 白丸 + 黒線の鍵アイコン。シンプルだが少しチープ。

具体策:
- 鍵アイコンの円を **わずかにグラデ**(`#ffffff` 上端 → `#f3f4f6` 下端)
- 鍵アイコン自体の線を **少し太く**(stroke-width 2.4 → 2.6)、内側に微細なハイライト
- 鍵のシャックル部分(U字)を **金色がかった色**(`#92400e` 等)にして「集めたら開く」を暗示
- セル下部に **「未取得」or「未獲得」マイクロラベル**(ink-500 + 9px) を控えめに加える案も可

## 守るべき制約(絶対変更禁止)

1. **ぽむのPNG画像** (`assets/pomu_pose_*.png`) を変更しない、画像生成しない
2. **HTML の主要構造**:
   - `<div class="sub-content" id="sub-collection">` 内の構成(`equipped-strip` → `filter-chips` → `collection-grid` → `collection-progress`)を維持
   - 装備中スロット 4 枠 (背景/フレーム/アバター/エフェクト)
   - グリッドセル 15個(順序・data-category・装備中/未取得状態)
3. **ガチャサブタブ(`#sub-gacha`)とショップサブタブ(`#sub-shop`)を一切変更しない**
4. **BottomNav 構造変更禁止**(順序: ホーム / ぽむタウン / 対戦 / フレンド / その他)
5. **既存 CSS 変数** (`:root`) を消さない(新規追加はOK)
6. **JavaScript 機能** を変えない:
   - フィルタチップの切替動作
   - セルクリック時の alert (locked/equipped/通常で分岐)
   - 装備中スロットのクリック alert
   - サブタブ切替、パックボタン、ガチャ履歴
7. **データ整合性**: 「38 / 120 種」「31.6%」の数値、各アイテム名、★レアリティの数は維持
8. **カラートークン**: sky-100/200, cloud-white/soft, pomu-yellow/orange, leaf-green, ink-900/500/300 を流用。新規色は最小限。

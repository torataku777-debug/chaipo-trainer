## 改善ポイント(優先度順)

### ★★★ 1. 包みの形状を「壺」から「巾着袋」へ

現状: 上下対称の楕円ベースで、見た目が「壺・甕」っぽい。  
目標: **巾着袋らしい躍動感** を出す。

具体的な要点:
- 上部の絞り口を **小さく** する(現状の幅広い楕円をやめて、しっかり絞った口に)
- 絞り口に **明確な紐結び** を追加(楕円の上に小さなボウタイ風のリボン、または十字に結んだ紐)
- 下部は **重力で垂れたシワ** を1〜2本入れる(布感)
- 全体のシルエットは「やや上が細く、下がぽってり」した洋ナシ型
- SVG path で `<path d="...">` を書き換えて実現
- 3パック(背景/フレーム/アバター)それぞれの色・柄・中央アイコンは維持

### ★★ 2. パックのCG質感・光と影

現状: 単色グラデで「フラット」な見た目。  
目標: **CGレンダー風の柔らかい陰影 + 軽いハイライト**

具体的な要点:
- 包み本体の上部に **白いハイライト**(楕円フィルタまたは光沢ライン)
- 下部に **やや暗い影** で立体感を出す
- ベースのリニアグラデーションを **ラジアルグラデーション**(光源左上想定)に変える
- 紐の影を本体に薄く落とす
- ぽむのPNGの足元に **柔らかい影** を追加(楕円 + ぼかし)
- パイナップル背景パックには **雲柄のテクスチャ強化**(現状の単純なドットより、ふわっとした雲)

### ★★ 3. ボタンの上質感

現状: ピンク・オレンジの単純グラデで、リリースレベルにはまだ物足りない。  
目標: ポケポケのボタンが持つ「触れる前に光ってる」上品な質感。

具体的な要点:
- 「無料で開ける」ボタンに **上端のハイライト+下端の暗影** で立体感
- ボタン全体に **微細な内側グロー**(ready 時のみ、外側の金グローと連動)
- 「100コイン」ボタンの枠を **二重線(外: 薄、内: 濃)** にして高級感
- コインアイコンをもう少し精密に(現状の text "¢" は手書き感)

### ★ 4. 全体の余白と密度

現状: パックカード3つで縦に並ぶが、密度が均一すぎ。  
目標: 1番目(is-ready)が **視覚的に重い** こと、2/3番目(locked)が **休んでる** ことが伝わる強弱。

具体的な要点:
- is-ready のカードを少しだけ **上下マージン拡大**(他のカードより主張)
- locked カードを **わずかに彩度低下**(現状の saturate 0.5 はやや強すぎ、0.7 程度に)
- カード間のスペーシングを 16px → 18px に微増

### ★ 5. 背景の白雲

現状: 楕円4個、opacity 0.75。やや単調。  
目標: もう少し本物の雲っぽく、立体感のある形状。

具体的な要点:
- 雲の輪郭を **複数楕円の重ね合わせ** で凹凸を作る
- 上部にやや明るい部分、下部に薄いグレー
- ただし主役を邪魔しないので opacity 0.6〜0.7、配置は控えめ

## 守るべき制約(絶対変更禁止)

1. **ぽむのPNG画像** (`assets/pomu_pose_*.png`) を変更しない。サイズ・位置の `<image>` 属性は調整可。
2. **5タブのBottomNav構造** を変えない(順序: ホーム / ぽむタウン / 対戦 / フレンド / その他)
3. **3パック構成と各パックの基本コンセプト**:
   - 背景パック(空ブルー、雲柄、ぽむ pose_6)
   - フレームパック(クリーム、格子柄、ぽむ pose_5)
   - アバターパック(パイナップル黄、葉柄、ぽむ pose_7)
4. **サブタブ構成**(ガチャ / ショップ / 所持) と切替ロジック
5. **状態クラス** (`is-ready` / `is-locked`) の動作。is-ready で揺れアニメ + 星明滅、locked で時計マーク
6. **カラートークン**(CSSの `:root` 変数)。新色を追加するのはOKだが、既存値を変えない
7. **JavaScript の機能挙動**(サブタブ切替、パック開封 alert、ガチャ履歴クリック)
8. **デバイスフレーム** 393×852、角丸 40px、影
9. **「ぽむ世界観を壊さない」**: アメコミ線画、グランジ、ダーク基調にしない。あくまで青空ライトトーン。

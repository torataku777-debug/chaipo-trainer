## 🏠 ホームタブ

**役割**: エンゲージメント特化。「今日もチャイポを起動して何ができるか」を一画面で。

1. ヘッダー: ぽむアバター + 段位 + シーズンレート / 通知・ログボアイコン
2. **PRIMARY CTA(動的切替)**:
   - 中断試合あり → 「続きから」
   - 中断なし → 「ランクマッチ」(シーズン残日数)
3. **ガチャ・リマインダー** ★ (Pokemon Pocket式デイリーフック)
   - 無料パック準備完了時: 3px枠 + 「!」赤バッジで強調
   - タップ → ぽむタウン → ガチャ画面に遷移
4. デイリーミッションカード
5. ログインボーナスカード
6. お知らせバナー(任意)

※モード選択は対戦タブで完結(機能重複回避)

---

## 🎁 ぽむタウンタブ(NEW)

**役割**: ガチャ・ショップ・コレクションを統合した「アイテムの場」。

サブタブ式で3画面切替:

### ガチャ(デフォルト)
- ヘッダー: 無料パック残り数 + リセットタイマー
- **ぽむパック × 3**(背景/フレーム/アバター)
- 各パック: 視覚 + 説明 + 無料パック状態 + 「無料で開ける」or「100コイン」
- ガチャ履歴へのリンク

### ショップ
- 特集バナー(シーズン限定パック等)
- カテゴリグリッド(背景/フレーム/アバター/エフェクト 各種類数表示)
- コイン・ジェム購入

### 所持(コレクション)
- フィルタチップ(全て / 背景 / フレーム / アバター / エフェクト)
- 3列グリッド表示
- 装備中バッジ
- タップで詳細・装備変更モーダル

## ⚔️ 対戦タブ(中央配置)

**役割**: モード選択専用画面。3段ヒエラルキー。

| モード | 階層 | アクション |
|---|---|---|
| ランクマッチ | 主役(3px枠) | タップで開始確認 → マッチング |
| カジュアル | 副 | タップで開始確認 → マッチング |
| AI対戦 | 補助 | モーダル: 2人/3人(SOON) |
| FL練習 | 補助 | モーダル: 14/15/16/17(EV表示) |

**注: フレンド対戦は含まない**(Q1決定)

## 画面遷移ルール

- **下部5タブナビは主要画面で常時表示**(例外: ゲームプレイ中・モーダル中)
- タブ内のサブ画面 → 左上の戻るボタンで親タブへ
- タブ間移動はナビタップのみ
- **モーダル**: 短い選択、確認
- **サブ画面**: 深いコンテンツ
- **サブタブ**: ぽむタウン内で使用(セグメントコントロール)

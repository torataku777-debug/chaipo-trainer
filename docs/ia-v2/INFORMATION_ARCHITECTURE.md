## 🏠 ホームタブ

**役割**: 「今日もチャイポを起動して何ができるか」を一画面で伝える。最も訪問頻度が高い。

### 構成(上から下へ)

1. **ヘッダー**: ぽむアバター + 段位 + シーズンレート / 通知・ログボアイコン
2. **PRIMARY CTA(動的切替)**:
   - 中断試合あり → 「続きから」
   - 中断なし → 「ランクマッチ」(シーズン残日数)
3. **SECONDARY ACTIONS**: カジュアル / AI対戦 / FL練習(横並び3つ)
4. **デイリーミッションカード**
5. **ログインボーナスカード**
6. **イベント・お知らせバナー**(任意)

※フレンド対戦は除外(フレンドタブで完結)

---

## ⚔️ 対戦タブ

| モード | 説明 | アクション |
|---|---|---|
| ランクマッチ | 段位上げる・シーズン進行 | タップで開始確認 → マッチング |
| カジュアル | レート変動なし | タップで開始確認 → マッチング |
| AI対戦 | 練習用 | モーダル: 2人/3人(SOON) |
| FL練習 | ファンタジー特訓 | モーダル: 14/15/16/17(EV表示) |

**注: フレンド対戦は含まない**(Q1決定)。

## 画面遷移ルール

- **下部4タブナビは主要画面で常時表示**(例外: ゲームプレイ中・モーダル中)
- タブ内のサブ画面 → 左上の戻るボタンで親タブへ
- タブ間移動はナビタップのみ
- **モーダル**: 短い選択(AI 2人/3人)、確認
- **サブ画面**: 深いコンテンツ(戦績、設定、ショップ)

---

## トーン統一方針(Q5: 後回しだが方針は確定)

- **全画面ライトトーン統一**(青空背景・白半透明カード)
- 設定・スタッツ・ショップの**ダーク/ベージュは廃止**
- 対戦動線完成後に着手

## 廃止する画面

| 廃止 | 理由 |
|---|---|
| CHAIPO TOWN(モード選択) | 4タブナビに分解吸収 |
| AI対戦選択(独立画面) | 対戦タブ内モーダル化 |
| FL練習選択(独立画面) | 対戦タブ内モーダル化 |
| オンライン対戦選択(独立画面) | 対戦タブ直下に統合 |
| フレンド対戦選択(独立画面) | フレンドタブに統合 |

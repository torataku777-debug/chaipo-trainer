## 改善ポイント(★優先度順)

### ★★★ 1. Featured バナーの渋谷夜景を精密化

現状: 縦長矩形のビルが9個並んだだけの単純な構成。窓の光は配置済みだがやや均一。

目標: 「シーズン1限定」感を出すリッチな夜景。

具体策:
- ビルの高さ・幅・形にバリエーションを増やす(上端を斜めにカット、最上部に小さなアンテナ等)
- 一部のビルに **電光看板の四角**(「ぽむ」「P」「カフェ」みたいな小さな矩形ロゴ風)を追加
- 窓の光の濃淡を増やす(消えている階、明るい階、薄い光)
- 月の周りに **薄い光輪**(radial gradient)
- 空に **星を数個**(小さな白点)
- 雲の代わりに薄い霧/もや(下部に薄いグラデ)
- 全体に Featured バナーの周りを **金色グロー** で強調

### ★★ 2. ボタン上質感を v3 同等に

現状: ショップ内のボタン(「購入する」、「購入」、カテゴリカード)が単純グラデ。

目標: v3 のガチャタブの「無料で開ける」「100コイン」ボタンと同レベルの質感。

具体策:
- 「購入する」(Featured バナー内): 上端の白ハイライト、下端の暗影、内側のソフトグロー、ホバー/タップでわずかなスケール
- 「購入」(コイン購入カード): 同上
- 全ボタンの border-radius・padding・font-weight・shadow を v3 のスタイルに揃える

### ★★ 3. カテゴリカードの CG 感

現状: 白カード + ソフト影だが、フラット感あり。

目標: v3 のパックカードと同等のCG感(微妙なハイライト、奥行き)。

具体策:
- 各カテゴリカードの上端に薄い白ハイライト
- 影をやや強める(立体感)
- タップ時のスケール・押下感を明確に
- アイコン背景の色枠は維持しつつ、その内側に薄いハイライト

### ★ 4. アイコン精細化

#### 配置エフェクト
現状: 大きな星 + 小さな星 1個。
目標: 「キラキラ集合感」。星を4〜5個に増やし、サイズと透明度でバリエーション。中心に大きな星 + 周囲に小さい星4個。

#### 背景
現状: 山と雲のシルエットが綺麗だが、月や手前の光が少ない。
目標: 山と雲はそのまま、雲の上に小さな鳥のシルエットを1つ追加、または月の輪郭を強調(任意)。

#### カードフレーム
現状: 角丸+点線でクラシックなフレーム。
目標: 内側の点線の代わりに、4隅の装飾的な飾り模様(◆ or ✦)を追加。

#### ぽむアバター(カテゴリアイコン)
現状: SVGで描いた小さなぽむシルエット。
目標: もう少しキャラシ準拠の精度を上げる(SVG path で目・くちばし・パイナップル葉のディテール)。  
※ **既存の PNG アセットを image として埋め込んでもよい**(36x36 サイズで pomu_pose_6.png を `<image>` 配置)。これが最もキャラシ準拠で楽。

### ★ 5. コイン購入カードの精密化

- コインアイコンとジェムアイコンを少し大きく
- ジェムに 3D感(光面と影面)
- カード自体に微細なハイライト

5. **JavaScript** の機能を変えない
6. ダーク背景・グランジは禁止(Featured バナーの夜景は **シーズン限定の特別感を出す例外**として認める)

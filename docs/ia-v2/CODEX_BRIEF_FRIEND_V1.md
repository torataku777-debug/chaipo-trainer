## 改善ポイント (★優先度順)

### ★★★ 1. プライマリ CTA「新しい部屋を作る」のヒエラルキー強化

現状: 黄色グラデの主役ボタンだが、その上のプロフィールカードのほうが視覚的に強い。「ここを押す」感が足りない。home v2 のランクマッチPRIMARYや more v2 のログアウトボタンとは異質。

具体策:
- ボタンの **高さ・パディングをわずかに増やす** (現状 padding 16px → 18px)
- **下端の inset shadow を厚くして「押せる」感** を強化 (`inset 0 -2px 0` を `-3px` に、色濃度も上げる)
- ボタン上端に **微細な金ハイライト** (現状の `inset 0 1px 0 rgba(255,255,255,0.78)` を二段に: `inset 0 1px 0 rgba(255,255,255,0.92), inset 0 2px 4px rgba(255, 235, 178, 0.62)`)
- アイコン (家+プラス) の **stroke-width を 2.4 → 2.6** に増やしてエンブレム感
- 文字サイズはそのまま (16px / weight 800)
- 全体に「ポムタウン三部作の主役ボタン (ガチャの『開く』など)」と統一感

セカンダリ「コードで参加」は現状維持で OK (主役を強調するため副は控えめのままがよい)。

### ★★★ 2. LIVE バッジの脈動と存在感

現状: 赤ピル + 内側ドット脈動。やや控えめ。

具体策:
- LIVE ピル本体に **外側 glow を追加** (`box-shadow: 0 0 0 0 rgba(239,68,68,0.0), 0 0 12px rgba(239,68,68,0.16)` などで常時の薄い光)
- 内側ドットの脈動半径を **現状 5px → 6-7px に拡大**、duration も `1.6s → 1.4s` で若干速める
- バッジ自体のフォント tracking を `0.10em → 0.14em` に強化して「英字バッジ」感
- 背景濃度: `--live-pink-soft` を `rgba(239,68,68,0.16) → rgba(239,68,68,0.20)` に
- 文字色はそのまま (`var(--live-pink)` = #ef4444)

### ★★★ 3. アクティビティ行の差別化と読みやすさ

現状: 4行とも色違いアバター (黄/シアン/ミント/ローズ) + 右側に小さなアクセントアイコン (星/トロフィー/雷/星)。情報密度は OK だが、「達成イベント名」がやや弱い。

具体策:
- イベント名の **色付き部分 (acc-event--cyan / mint / rose)** をやや太く・パディングを取り、「タグ風」に小さく寄せる
  - 例: 背景色を transparent → そのカラーの極薄 (`rgba(...,0.10)`)、padding `1px 5px`、border-radius `5px`
  - 「AAでFL達成」「プラチナ 4 に昇格」「スクープ達成」「KKでFL達成」が小タグとして際立つ
- アバターの直径を **36px → 38px**、内側 inset shadow を 1px 強める (more v2 のプロフィールアバターと統一感)
- 右側アクセントアイコン:
  - 星 (FL系) → 塗り強め (opacity 0.5 → 0.7)、色を `var(--pomu-yellow)` 寄りに変更
  - トロフィー (昇格) → 塗り `var(--pomu-yellow)`、`opacity 0.7`
  - 雷 (スクープ) → 塗り `var(--leaf-green)`、`opacity 0.7`
- 行間 (padding) を `12px 14px → 13px 14px` で 1px だけ余裕を出す

### ★★ 4. 対戦中バッジ ⚔ の質感統一

現状: フレンドリスト 1 行目「ぽむさん子」の status 欄に `.battle-pill` (赤ピル + ⚔ 絵文字)。LIVE バッジと色が同じだが、絵文字フォントがOSによってばらつく。

具体策:
- ⚔ を **SVG アイコンに置換** (LIVE ピルと同じスタイル、12×12 サイズ、stroke-width 2)
- 背景濃度を LIVE と同じく `rgba(239,68,68,0.20)` に揃える
- 内側にドット脈動は不要 (LIVEと差別化、対戦中は静的でよい)
- 「対戦中」テキストは維持

### ★★ 5. シェアボタンの押下感と視認性

現状: 36×36 のガラス調ボタン、シアン文字 (`--title-cyan-fg`)。クリーンだが「押せる」サインが弱い。

具体策:
- 通常状態の **背景濃度をやや上げる** (`linear-gradient(180deg, var(--cloud-soft) 0%, var(--cloud-white) 100%)` の白寄りグラデを少し主張)
- アイコン (↗) の stroke-width を `2 → 2.2` に増やす
- ホバー / アクティブ時のハイライトを `rgba(143, 200, 224, 0.16)` の薄シアン背景に切り替え
- 押下時 (`:active`) の **シャドウ反転 + 1px沈み込み** を強める

### ★★ 6. フレンドリストの密度と境界線

現状: 6 行、各 12px×14px パディング。各行間に `--hairline` 境界線。やや密で、長い名前だと窮屈に見える可能性。

具体策:
- 各行のパディングを **`12px 14px → 13px 16px`** に微増 (上下1px、左右2px)
- アバターのサイズはそのまま (40px)
- 対戦招待ボタン (黄色丸) の **shadow を強めて主役感** (友達がオンラインの時の「いますぐ対戦」誘導):
  - 現状: `0 2px 5px rgba(245, 158, 11, 0.22)` → `0 4px 10px rgba(245, 158, 11, 0.28), 0 0 0 1px rgba(146, 64, 14, 0.14)`
  - inset highlight を追加: `inset 0 1px 0 rgba(255,255,255,0.86)`
  - ホバー時に **微細にスケール+ハロー** (`transform: scale(1.04); box-shadow に外側 ring`)
- 対戦招待ボタンのアイコンを **「⚔ (剣)」SVG に変更** が望ましい (現状は対角矢印で「フレンドに送る」のように見える)
  - 推奨: 上記4と同じ剣SVG (stroke 2.4、白文字)

### ★ 7. 対戦履歴の WIN/LOSE バッジの可読性

現状: WIN は薄緑背景 + 緑文字 / LOSE は薄赤背景 + 赤文字。スコア表示 (+18 / −7 / +12) は tabular-nums で揃っている。

具体策:
- バッジ枠線を **`border` から `box-shadow inset` に変更** で内側に光沢
  - WIN: `box-shadow: inset 0 0 0 1px rgba(21, 128, 61, 0.32), inset 0 1px 0 rgba(255,255,255,0.6)`
  - LOSE: `box-shadow: inset 0 0 0 1px rgba(190, 18, 60, 0.22), inset 0 1px 0 rgba(255,255,255,0.6)`
- スコア部 (`+18`/`−7`/`+12`) の font-weight を `800 → 900`、letter-spacing をやや詰める
- バッジ全体の border-radius を `11px → 12px` に揃える

### ★ 8. プロフィールカードのバッジ間 gap

現状: 段位バッジ・レートバッジ・称号バッジが横並びで gap 8px。「FL職人」のシアンが more v2 と同じ配色で揃っている。

具体策:
- gap を `8px → 9px` に
- 段位バッジ「プラチナ 3」内のスター SVG の **塗りを濃く** (現状 `fill="currentColor"` でゴールド、`drop-shadow` を 1px 強めて立体感)
- 各バッジ縦パディングを **`5px 10px 5px 8px → 6px 11px 6px 9px`** に微増 (バッジが少し大ぶりになる)

### ★ 9. SVG アイコンの統一パス

具体策:
- 全 SVG の stroke-width を **`2` に統一** (現状: 2 / 2.2 / 2.4 が混在)
  - 例外: 主役級 (cta-primary の家アイコン, 対戦招待の剣) のみ 2.4 で OK
- アクティビティ右側アイコン (現状 fill+stroke 混在) を **fill 系に統一** (アクセント色強調)
- フレンド追加アイコン (header-action) は現状の人物+ プラスマーク維持、stroke 2 に

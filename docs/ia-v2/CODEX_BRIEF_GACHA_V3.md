## カード束のデザイン仕様

### 共通の構造

各パックの SVG `<g class="pack-bundle">` 内を、巾着袋の path から下記の「クラシックなカード束」に書き換える:

```
- 5〜6枚のトランプカードが扇形に少し広がって重なっている
- 最前面のカード(主役): 中央やや上に配置、ほぼ垂直
- 後ろのカード: 左右に±5〜12度ずつ角度をつけて顔を覗かせる
- 各カードのサイズ: viewBox(104×132) に収まる形で、最前面カードが約 52×72px
- カードのアスペクト比: 約 2.5:3.5(トランプの一般比率)
- 角丸: 5〜6px
- カード同士の影で奥行きを出す
```

### 最前面カード(主役)のデザイン

- **背景色とパターン**: パックごとに違う(下記表)
- **中央のアイコン**: パック内容を象徴する小さなシンボル(下記表)
- **クラシックトランプ枠**: 内側に薄い線で囲んだ二重枠(任意、品の良さ)
- **わずかな光沢ハイライト**(v2のCG感を引き継ぐ)

### 3パックの差別化

| パック | カード裏面色 | パターン | 中央アイコン | ぽむポーズ |
|---|---|---|---|---|
| 背景パック | 空ブルー(#c5e2f0 → #8fc8e0グラデ) | 控えめな雲柄(現状の `bgPackCloud` パターン流用) | 山と雲のシルエット | pose_6 |
| フレームパック | クリーム(#fef3c7 → #fde68a) | 細格子(現状の `frPackGrid`) | 角丸フレーム枠 | pose_5 |
| アバターパック | パイナップル黄(#fcd34d → #f59e0b) | 葉柄(現状の `avPackLeaf`) | ぽむシルエット(小・白丸+パイナップル) | pose_7 |

### 後ろのカード

- 最前面と同じ裏面デザイン
- ただし扇形に少しずつズレて表示される(4辺だけ覗く)
- 透明度や明度をわずかに下げて「奥」感を出す

### 帯/リボン(任意)

- 必須ではない。クラシックさを優先するなら **無し** を推奨
- もし入れるなら、カード束の中央付近に細い帯(パックごとに色違い、文字なし)

### ぽむの配置

- v2 と同じ位置(`x="46" y="64" width="62" height="62"`)
- カード束の右下から覗く構図
- ぽむのPNGは絶対に変更しない

### 影とCG感

- カード束全体の底に柔らかい楕円影(v2 と同等)
- 各カード間に薄い影(後ろのカードが最前面の影に落ちる感)
- 最前面カードの上端に白いハイライトライン

### アニメーション

- v2 と同じ揺れアニメ(2秒周期 ±1.6deg)を `.pack-bundle` に適用維持
- 星マーク明滅も維持
- タップ時の強揺れ(JS)も維持

## 改善ポイント(★優先度順)

### ★★★ 1. PRIMARY CTA の磨き

現状: 紫グラデの大きいカード(対戦専用色として認める)。但し細部の質感が物足りない。

具体策:
- 紫グラデを保ちつつ、**カード全体のCG感を強化**(上端の白ハイライト、左下の暗めの影、内側のソフトグロー)
- 装飾円(`cta-deco-1`, `cta-deco-2`)をもう少し本物の光源風に(粒状感やスターバースト)
- 右下の **黄色い矢印ボタン**を大きく強調(直径 +4-6px、影を深く、内側ハイライト)
- 「RANK MATCH」「ランクマッチ」テキストの **文字シャドウ強化**(可読性UP)
- **下端に薄いハイライトライン**(光沢)
- 「→ 中断ありver を見る」リンクのデザイン微調整(背景に半透明ピル形)

### ★★ 2. ぽむアバターの枠

現状: 円形 + ゴールドの3pxリング + 白の2pxインナーリング。やや太い。

具体策:
- ゴールド枠を **2pxに**、または **ゴールド + 薄いオレンジ二重リング** にして上品に
- フレーム外周に **微細なドロップシャドウ**で立体感
- ぽむ画像の表示エリアの **背景に薄い空ブルー(オフホワイト)** を入れて柔らかく
- ぽむ自体の縁取り線が背景に溶け込まないよう、コントラスト確保

### ★★ 3. ガチャリマインダーのカード束SVG精細化
